# Introduction to SQL

## Overview
This project contains beginner SQL queries and database operations completed while learning SQL.

The project demonstrates how to:
- Create a database
- Create tables
- Insert data into tables
- Retrieve data using SELECT
- Filter records using WHERE
- Sort records using ORDER BY

## Database Structure
The project includes tables such as:
- EmployeeDemographics
- EmployeeSalary

## SQL Concepts Used
### 1. CREATE DATABASE
Used to create a new database.

### 2. CREATE TABLE
Used to create tables and define columns with data types.

### 3. INSERT INTO
Used to add records into the tables.

### 4. SELECT
Used to retrieve data from tables.
